# Cafe Website

A professional and responsive website for a café, built using **React + Vite**.  
Designed to showcase the café’s menu, location, gallery, and contact information. Mobile-friendly and ready to deploy.

## Features

- Fully responsive design for desktop and mobile
- Smooth scrolling with sticky shrinking navbar
- Hero section with overlay and CTA button
- Menu section with clear layout
- Gallery with images
- Embedded Google Map for location
- Contact section with phone and hours
- Footer with developer credit

## Tech Stack

- React
- Vite
- HTML, CSS, JavaScript

## Deployment

This project can be deployed on **Vercel**:

1. Push the project to GitHub
2. Import repository in Vercel
3. Deploy with default React/Vite settings
